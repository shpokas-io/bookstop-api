### BookSpot API

A simple and efficient backend API for managing books and reservations for the BookSpot project. Built with ASP.NET Core, this RESTful API allows users to perform CRUD operations on books and reservations.

### Tech used:

- ![ASP.NET Core][ASP.NET Core]
- ![Entity Framework Core][Entity Framework Core]
